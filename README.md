
# Food choices and preferences of college students

## Project Overview:
### This dataset includes information on food choices, nutrition, preferences, childhood favorites, and other information from college students. This project analyzes survey data from 126 college students to identify psychological, economic, and background factors influencing dietary habits and food preferences. Utilizing Excel and Power BI, the analysis investigates correlations between lifestyle factors and nutritional choices, while transforming raw data into actionable insights for student wellness. The project specifically addresses data cleaning challenges, such as handling missing values and unstructured text from the original survey.

## Key Insights :
### Analysis of this dataset college students often prioritize food cost over nutritional value and experience increased comfort food consumption linked to higher stress levels. Key findings also highlight significant gaps in caloric awareness and distinct gender-based preferences regarding healthy eating and convenience.
